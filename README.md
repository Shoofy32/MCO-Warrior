# Warrior (MCO)

Turn-based combat built with Java where players can choose different equipment to battle with, a variety enemies to fight, 
and various environments that affect the gameplay.

---

## Features

- Turn-based combat system
- Includes attacking, defending, charging, consuming, and skill mechanics
- Selection system where players can choose weapons, armor, consumables, enemies, and environments
- Enemies have their own turn logic, with some having their own equipment similar to the user.
- Consumables provide a variety of buffs or nerfs to the Player and/or Enemy
- Enchanted weapons that have built-in skills that can be activated after performing the skill requirement
- Includes Graphical User Interface (GUI) built with Java Swing
- Pixelated art made via Aseprite

---

## Installation and Running 

1. Download required files
2. Open CMD in project root directory and type the following commands to run the program:
```bash
javac *.java
java Driver
```
